本项目的目标不是随机生成动物形态，而是在最大可操控性的前提下，尽可能逼近 reference 文件夹中的参考图结构。所有算法与规则都服务于这一目标，其余结构机制应适可而止，不追求不必要的复杂度。

系统的核心思想是“定位优先”。形态生成的第一步不是生成圆或椭圆，而是先通过三分正方形网格进行结构定位。网格的作用只是确定关键结构的大致位置，例如身体中心、头部位置、关节位置等。网格最多进行三次分形：3×3 → 9×9 → 81×81。分形不是全局同时进行，而是只在需要精细定位的区域进行，其余区域保持原结构。网格只负责定位，不负责最终形态。

在定位完成后，在对应网格位置上放置圆。每一个圆都有一个圆心 O，这些圆心对应动物结构中的关键节点，例如身体中心、头部、关节等。圆的主要作用是确定结构中心点。

圆确定之后，可以进行向外扩散的涟漪式运动。圆从圆心开始逐渐扩大，类似水面涟漪，直到达到结构边界或预设切线位置时停止。扩散速度应平缓，使视觉上可以清晰识别扩散过程。

当圆结构确定之后，再使用椭圆进行形态构型。椭圆用于描述身体、四肢、头部等结构。椭圆通常来自圆的几何关系，例如圆的直径可以作为椭圆的长轴，椭圆内部也可以继续生成更小的圆，再生成新的椭圆，从而形成有限的嵌套结构。但这种嵌套只用于必要的形态描述，不需要无限递归。

整体生成流程应始终遵循以下顺序：
网格定位 → 圆心确定 → 圆扩散 → 椭圆构型 → 最终动物形态。

系统生成的结果需要尽可能接近 reference 文件夹中的参考图片结构。参考图是系统生成的目标模板，所有定位、圆结构和椭圆构型都应围绕这些参考结构进行调整。

圆—椭圆变化规则
1.每一个结构节点首先是一个圆，圆提供中心点 O 和结构尺度。
2在一个圆内部可以生成一个椭圆：椭圆的中心与圆心 O 相同，椭圆的长轴等于圆的直径。
3.椭圆可以旋转，用来表达结构的方向（如身体、腿、颈等）。
4.在一个椭圆内部可以继续生成一个新的圆：新圆的直径对齐椭圆的短轴。
5.因此结构关系是一个递归链：圆 → 椭圆 → 圆 → 椭圆 → …
6.圆负责确定位置和尺度，椭圆负责生成形态和方向。
7.这种递归可以持续，但在实际结构中应适可而止，以保持结构简洁。
