请先阅读 rules.txt，然后分析 reference 文件夹中的图片。

reference 文件夹中的每个动物都有三个图层：

layer1_combined
包含动物形态、圆结构和网格系统的综合图。

layer2_circle_system
只包含构成动物结构的圆。

layer3_grid_system
只包含用于定位的三分正方形网格。

在分析这些图片时，请重点识别以下内容：

1
圆心 O 的位置。
每一个圆都有一个圆心 O，这些圆心对应动物结构中的关键节点，例如身体中心、头部位置、关节位置等。

2
不同圆的直径层级。
识别哪些圆是最大结构（通常是身体），哪些是中等结构（例如头部或翅膀），哪些是小结构（例如脚或尾巴）。

3
网格与圆心的关系。
判断圆心是如何落在三分网格中的。

4
圆与椭圆的关系。
动物形态中的椭圆通常来自这些圆的几何关系。

在完成分析之后，请总结：

每个动物的
- 圆心数量
- 主要结构圆
- 次级结构圆
- 小结构圆

不要立即生成代码。
先给出结构分析结果。
